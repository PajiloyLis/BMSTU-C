# Тесты для лабораторной работы №01

## Входные данные
Целые положительные a, d

## Выходные данные
Целые неотрицательные q, r - частное и остаток от деления a на d соответственно

## Позитивные тесты:
- 01 - a делится нацело на d
- 02 - Обычный тест
- 03 - d > a, частное равно 0
- 04 - a == d

## Негативные тесты
- 01 - a < 0
- 02 - d < 0
- 03 - a и d отрицательные
- 04 - вместо a буква
- 05 - вместо d буква
